
# Random Quote Generator

**Автор**: Матвей Фомин

## Описание

Приложение генерирует случайные цитаты из предопределённого списка, сохраняет историю всех сгенерированных и вручную добавленных цитат в JSON-файл. Позволяет фильтровать историю по **автору** и Random Quote Generator. Встроена проверка на пустые строки при добавлении новых цитат.

## Функциональность

- Генерация случайной цитаты из встроенного списка.
- Отображение текущей цитаты и её автора.
- История всех цитат (с датой) в таблице.
- Фильтрация истории по автору (частичное совпадение, регистронезависимо).
- Фильтрация по теме.
- Добавление новых цитат пользователем (сохраняются в историю).
- Сохранение истории в `quotes.json` и загрузка при запуске.
- Графический интерфейс (tkinter).
